javaのfor文について解説する前に、 まずはfor文とは何かについて解説すると、 同じことを何度も繰り返す手順のことをfor文と呼び、 大抵のプログラミング言語というパソコンを奴隷のように扱う命令言語の全てに含まれています。 人間は基本的には同じことを何度も繰り返すのが嫌いなわけで だからこそ同じ処理に対して回数を指定して嫌いなことを避けているのです。 例えばですが 「一匹残らず駆逐してやる」 を5回くりかえしたい場合 for文がない世界線では 
「一匹残らず駆逐してやる」

System.out.println("「一匹残らず駆逐してやる」");
となりますが、 for文がある世界線では 
「一匹残らず駆逐してやる」　を五回繰り返せ

for (int i = 0; i < 2; i++){
  System.out.println("「一匹残らず駆逐してやる」");
}
と簡単に記述できるのがfor文の強みなのです ところで、プログラミングと聞くとみなさん難しく構えがちですが、 大抵のソフトウェアと呼ばれる頭良さげな機構は、 この繰り返しを担当するfor文,if文,関数だけから構成される場合がほとんどで、 それを一人ではなく大勢の人数でやろうとした結果複雑なよくわかんない手法や横文字が出てきたわけです。 
よくわかんない手法や横文字の例
git,
linux,
統合開発環境
アジャイル,
ウォーターforル,
DDD,
話をfor文に戻しますが、 javaのfor文には普通の書き方と怠惰な書き方があります。 基本的な書き方は先ほど紹介した通り,指定回数とカウンターが必要になるわけですが, 
for (int i = 0; i < 5; i++){
  System.out.println("「一匹残らず駆逐してやる」");
}
例えば 「一匹残らず駆逐してやる」セットがあらかじめ5セット用意されていた場合、 そのセットから一つずつ取り出した方が楽なわけです。 それを実現するのが怠惰な書き方なわけで 今回の例ですと、 
「一匹残らず駆逐してやる」お得5セットパック

for (セリフを入れる箱 ; 「一匹残らず駆逐してやる」お得5セットパック){
    System.out.println(セリフを入れる箱);
}
となります。 以上。javaのfor文について3分で解説する試みでした。 title:「java for 書き方」で調べた人へ description:javaのfor文の書き方について簡単な解説を行います。
