git clone https://github.com/ユーザー名/リポジトリ名.git

git clone https://github.com/kawadasatoshi/techblog.git

##その4　編集作業を行う ##その5 git add .で編集したファイルをgitの管理ファイルとして設定 

git add .
このコマンドは基本何も出力しません。 ##その6 コミットする 

git commit -m "test"
実行結果例 

minegishiminami@minegishiminami-ThinkPad-X1-Yoga-4th:~/myworking/techblog$ git commit -m "test"
[main 7018e6d] test

##pushでいよいよgithubにアップロード 

git push origin master
その際に以下のような表示が出ることがある その際はgithubに設定したメールアドレスとユーザー名を設定しましょう 

  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
